# Dark-Light Mode PHP

A simple PHP exercise that lets users switch between light and dark themes. The chosen theme is stored in a cookie and automatically applied on reload.

## Features
- Light/Dark mode   
- Saves user preference with cookies  
- Dynamic CSS loading based on the saved theme  

## Usage
1. Open `mode.php` on your local server.  
2. Choose your theme and click **Apply theme**.  
3. The page reloads and remembers your choice.  

## Files
- `mode.php` → main logic  
- `light.css` → light theme  
- `dark.css` → dark theme
